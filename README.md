# Technomancy Visualized

![Screenshot from the application's main UI, showing a network of interconnected nodes.](https://hosting.photobucket.com/bbcfb0d4-be20-44a0-94dc-65bff8947cf2/642684d8-99b2-4b42-9125-84b655d87c2b.png)

Interlinks ideas about technology, consciousness, cyberculture and magick to explore how humans and machines co-evolve meaning, power and awareness in the global knowledge economy.

## Overview

An interactive, force-directed knowledge graph built with React, TypeScript and D3 which visualizes an ecosystem of concepts and resources spanning technology, cyberculture, consciousness studies, magick, transhumanism and open-source philosophy. Idea nodes are connected to tag nodes representing shared themes, allowing users to explore relationships through highlighting and emphasis. Hovering over idea nodes reveals descriptions in a tooltip, while clicking them opens a side panel with descriptions and external links where available. The result is an interface for mapping intersections between technomancy, systems theory, cybernetics, psychedelics, open culture and emerging posthuman thought.

## Set Up Instructions

Below are the required software programs and instructions for installing and using this application on a Linux machine.

### Programs Needed

- [Git](https://git-scm.com/downloads)

- [Node.js](https://nodejs.org/en)

### Steps

1. Install the above programs

2. Open a terminal

3. Clone this repository using `git` by running: `git clone git@github.com:devbret/technomancy-visualized.git`

4. Navigate to the repo's directory: `cd technomancy-visualized`

5. Install the frontend: `npm install`

6. Launch the frontend: `npm run dev`
